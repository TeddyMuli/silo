## Description
Silo a modern solution for all your cloud storage needs.

## Requirements
1. Go
2. NodeJS
3. Redis
4. Postgres

## Installation
```bash
git clone https://github.com/TeddyMuli/silo.git
```

### Backend
1. Navigate to the `server` directory.
```bash
cd server
```

2. Install Packages
```bash
go mod tidy
```

3. Run the app
```bash
go run main.go
```

### Frontend
1. Navigate to the `web_client` directory.
```bash
cd web_client
```

2. Install dependencies
```bash
npm install
```

3. Run the app.
```bash
npm run dev
```
